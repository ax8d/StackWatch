StackWatch settings

Free — improves rate limits for server-software CVE lookups (Apache, nginx, PHP, IIS, WordPress core). Get one at nvd.nist.gov/developers/request-an-api-key. Hidden by default since it's a credential — click Show to reveal it.

Two things, both opt-in together: (1) checks a small set of known file paths (like /wp-login.php or /readme.html) on the site you're visiting to confirm CMS/version details headers alone can't reveal, and (2) fetches a same-site CSS/JS asset file to read its leading version comment when a library's filename and headers don't expose a version. Both send a few extra requests to that site's server — some site owners may log this as scanning activity.
